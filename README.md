# MetroTech メトロテック

> Recovered transmissions from a band that spent the voyage home remembering.

MetroTech is an audio-visual fiction: three young women aboard an evacuation city-ship on the long voyage back to Earth, recording music on aging analog equipment and transmitting it on a dead channel. The albums are those recovered transmissions.

Tape-worn. Analog. Slow. Dreaming.

**Live site:** [metrotechmusic.github.io](https://metrotechmusic.github.io)
